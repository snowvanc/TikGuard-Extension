1. 项目简介 (The Problem & The Solution)

“在经历了数次因为网络环境波动和无意间的违禁词导致 TikTok 店铺被封后，我开发了这款名为 TikGuard 的 Chrome 插件”

2. 核心功能 (Features)

实时网络环境与时区监控 (Geo-Fence)

本地与云端同步的违禁词拦截引擎 (支持输入框实时检测)

基于 Cloudflare Workers 的订阅鉴权机制

集成大语言模型进行文案合规重写

3. 技术架构 (Tech Stack)
说明前端是 Chrome Extension V3，后端是 Cloudflare Workers，支付对接了爱发电 (Afdian)。

4. 为什么开源？(The Post-Mortem)

“项目历时半年开发，跑通了技术和商业闭环，但由于在移动端引流至桌面端产品时遇到了巨大的转化鸿沟，以及安全工具本身的信任门槛，最终未能实现商业化。现将全套代码开源，作为这段经历的纪念，也希望能给其他做跨境电商辅助工具的开发者提供一些参考。”
