# TikGuard - Chrome Extension

TikGuard is a powerful browser extension designed for cross-border e-commerce sellers. It helps users manage store operations efficiently and safely.

## Core Features
* **Risk Scanner**: Automatically detects sensitive words and prevents policy violations in input fields.
* **IP Monitor**: Real-time network environment monitoring to avoid account association risks.
* **Image Scrubber**: Built-in tool to process images and prevent duplicate content detection.
* **AI Assistant**: Quick text polishing and SEO title generation.
